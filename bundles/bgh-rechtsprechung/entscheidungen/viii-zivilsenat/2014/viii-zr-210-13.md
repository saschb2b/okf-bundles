# Leitsatz

Erteilt der Vermieter dem Mieter eine Erlaubnis zur Untervermietung, so kann dieser ohne besondere Anhaltspunkte nicht davon ausgehen, dass die Erlaubnis eine tageweise Vermietung an Touristen umfasst.

# Normen

- § 553 BGB
