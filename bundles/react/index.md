# React

Agent-readable knowledge of React, transformed from the official documentation at [react.dev](https://react.dev). One concept per documentation page: the Learn guide (how React works and how to think in it) and the full API Reference (exact signatures, parameters, returns, and caveats). Each concept's `resource` is its canonical react.dev URL, and the body is a structured, transformed summary rather than a copy.

Start here: [Overview](overview.md).

- [Quick start and tutorial](get-started/index.md) - A fast tour of the core concepts, the tic-tac-toe tutorial, and Thinking in React.
- [Installation](installation/index.md) - Start a new React app, build from scratch, or add React to an existing project.
- [Setup](setup/index.md) - Editor setup, TypeScript, and React Developer Tools.

# Learn React

The four core sections of the Learn guide, in reading order.

- [Describing the UI](describing-the-ui/index.md) - Components, JSX, props, conditional rendering, lists, and keeping components pure.
- [Adding Interactivity](adding-interactivity/index.md) - Events, state, the render-and-commit cycle, and updating objects and arrays in state.
- [Managing State](managing-state/index.md) - Structuring state, lifting it up, reducers, and context.
- [Escape Hatches](escape-hatches/index.md) - Refs, Effects, when not to use an Effect, and custom Hooks.
