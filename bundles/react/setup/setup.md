# Summary

React integrates with editors, TypeScript, browser extensions, and compilers. This section orients you to the tooling that surrounds day to day React work and points to the dedicated guide for each topic.

# In this section

- [Editor Setup](editor-setup.md): recommended editors and how to configure linting and formatting to work with React.
- [Using TypeScript](typescript.md): add type definitions to a React project and type your components and Hooks.
- [React Developer Tools](react-developer-tools.md): browser extension to inspect components, edit props and state, and find performance problems.
- React Compiler: a tool that automatically optimizes your React app. See [/react-compiler/react-compiler.md](/react-compiler/react-compiler.md).
