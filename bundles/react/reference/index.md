# API Reference

The complete React API surface, one concept per documented export, mirroring the Reference section of react.dev. Where the Learn guide teaches the ideas, this reference gives the exact signatures, parameters, return values, and caveats.

- [react](react/index.md) - The core package: Hooks, built-in components, top-level APIs, and legacy APIs.
- [react-dom](react-dom/index.md) - Rendering to the browser DOM and to HTML on the server: components, hooks, client, server, and static APIs.
