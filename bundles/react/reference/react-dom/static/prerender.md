# Reference

`prerender` renders a React tree to static HTML using a Web Stream. It is a static site generation (SSG) API that waits for all data to load before resolving, including Suspense-fetched data.

```js
import { prerender } from 'react-dom/static';

const { prelude, postponed } = await prerender(reactNode, options?)
```

Runtime: Web Streams API (Deno, modern edge runtimes). For Node.js, use [prerenderToNodeStream](/reference/react-dom/static/prerenderToNodeStream.md) instead.

## Parameters

- `reactNode`: The React node to render to HTML, for example `<App />`. It is expected to represent the entire document, including the root `<html>` tag.
- `options` (optional): An object with static generation options.
  - `bootstrapScriptContent`: A string placed in an inline `<script>` tag.
  - `bootstrapScripts`: An array of string URLs for `<script>` tags to emit on the page.
  - `bootstrapModules`: Like `bootstrapScripts`, but emits `<script type="module">`.
  - `identifierPrefix`: A string prefix for IDs generated by [useId](/reference/react/hooks/useId.md). Must match the prefix passed to [hydrateRoot](/reference/react-dom/client/hydrateRoot.md).
  - `namespaceURI`: Root namespace URI. Defaults to HTML. Use `'http://www.w3.org/2000/svg'` for SVG or `'http://www.w3.org/1998/Math/MathML'` for MathML.
  - `onError`: Callback fired on any server error, recoverable or not. Defaults to `console.error`.
  - `progressiveChunkSize`: The number of bytes in a chunk.
  - `signal`: An AbortSignal to abort prerendering.

## Returns

A Promise that resolves on success to an object with:

- `prelude`: A Web Stream of HTML. Send it in chunks or read it entirely into a string.
- `postponed`: A JSON-serializable, opaque object passable to [resume](/reference/react-dom/server/resume.md) if `prerender` did not finish. `null` if `prelude` contains all content and no resume is necessary.

On failure, the Promise rejects. Use that to output a fallback shell.

# Usage

- Render to a response stream. React injects the doctype and bootstrap `<script>` tags.

```js
async function handler(request) {
  const { prelude } = await prerender(<App />, {
    bootstrapScripts: ['/main.js']
  });
  return new Response(prelude, {
    headers: { 'content-type': 'text/html' },
  });
}
```

- Render to a string by reading the `prelude` stream to completion.

```js
const reader = prelude.getReader();
let content = '';
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  content += Buffer.from(value).toString('utf8');
}
```

- Wait for all data. `prerender` only suspends on Suspense-enabled sources: Suspense-enabled frameworks (Relay, Next.js), code lazy-loaded with [lazy](/reference/react/apis/lazy.md), and Promises read with [use](/reference/react/apis/use.md).
- Abort after a timeout with an `AbortController` passed as `signal`. The `prelude` contains HTML prerendered before the abort, with incomplete Suspense boundaries in their fallback state. Continue later with [resume](/reference/react-dom/server/resume.md) or [resumeAndPrerender](/reference/react-dom/static/resumeAndPrerender.md).

# Caveats

- `nonce` is not available when prerendering. Nonces must be unique per request, so including one in prerendered content would be insecure under CSP.
- Does not stream as content loads. It waits for the entire app to finish before resolving. To stream, use [renderToReadableStream](/reference/react-dom/server/renderToReadableStream.md).
