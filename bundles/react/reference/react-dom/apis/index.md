# React DOM APIs

- [createPortal](createPortal.md) - Renders children into a different DOM node while keeping them part of the React tree.
- [flushSync](flushSync.md) - Forces React to flush updates inside a callback synchronously so the DOM updates immediately.
- [preconnect](preconnect.md) - Asks the browser to open a connection to a server you expect to load resources from.
- [prefetchDNS](prefetchDNS.md) - Asks the browser to resolve the DNS of a server you expect to load resources from.
- [preinit](preinit.md) - Eagerly fetches and evaluates a stylesheet or external script.
- [preinitModule](preinitModule.md) - Eagerly fetches and evaluates an ESM module.
- [preload](preload.md) - Eagerly fetches a resource such as a stylesheet, font, or script without evaluating it.
- [preloadModule](preloadModule.md) - Eagerly fetches an ESM module without evaluating it.
