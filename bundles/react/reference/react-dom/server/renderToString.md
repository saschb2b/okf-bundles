# Reference

`renderToString` renders a React tree to an HTML string on the server. It is a legacy, blocking API with limited capabilities. Prefer the streaming or prerender APIs.

```js
import { renderToString } from 'react-dom/server';

const html = renderToString(reactNode, options?)
```

Runtime: Server-side (Node.js, Deno, edge runtimes). It also runs in the browser, but should be avoided on the client.

## Parameters

- `reactNode`: The React node to render to HTML, for example `<App />`.
- `options` (optional): An object for server render.
  - `identifierPrefix`: A string prefix React uses for IDs generated by [useId](/reference/react/hooks/useId.md). Useful to avoid conflicts when using multiple roots on the same page. Must be the same prefix passed to [hydrateRoot](/reference/react-dom/client/hydrateRoot.md).

## Returns

An HTML string.

# Usage

- Render to a string and send it as the server response. On the client, call [hydrateRoot](/reference/react-dom/client/hydrateRoot.md) to make the HTML interactive.

```js
app.use('/', (request, response) => {
  const html = renderToString(<App />);
  response.send(html);
});
```

- To render a component to HTML on the client instead, use [createRoot](/reference/react-dom/client/createRoot.md) with [flushSync](/reference/react-dom/apis/flushSync.md).

```js
const div = document.createElement('div');
const root = createRoot(div);
flushSync(() => {
  root.render(<MyIcon />);
});
console.log(div.innerHTML); // "<svg>...</svg>"
```

# Caveats

- Limited Suspense support. If a component suspends, `renderToString` immediately sends its fallback as HTML.
- Does not support streaming or waiting for data.
- Works in the browser but is not recommended in client code.

# Recommended alternatives

- Streaming, Node.js: [renderToPipeableStream](/reference/react-dom/server/renderToPipeableStream.md).
- Streaming, Web Streams: [renderToReadableStream](/reference/react-dom/server/renderToReadableStream.md).
- Static prerender, Node.js: [prerenderToNodeStream](/reference/react-dom/static/prerenderToNodeStream.md).
- Static prerender, Web Streams: [prerender](/reference/react-dom/static/prerender.md).
