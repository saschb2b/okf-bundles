# React Server Components

Server Components, Server Functions, and the two directives that draw the server/client boundary.

- [Server Components](server-components.md) - Components that render ahead of time on the server or at build time, separate from the client bundle.
- [Server Functions](server-functions.md) - Async functions that run on the server and can be called from Client Components, often as form actions.
- ['use client'](use-client.md) - Directive that marks a module and its dependencies as client code, creating the server/client boundary.
- ['use server'](use-server.md) - Directive that marks server-side async functions as callable from the client.
