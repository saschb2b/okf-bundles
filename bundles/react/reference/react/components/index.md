# React Built-in Components

- [Fragment](Fragment.md) - Group elements without a wrapper DOM node, optionally with a key or a ref.
- [Profiler](Profiler.md) - Measure the rendering performance of a tree programmatically via an onRender callback.
- [StrictMode](StrictMode.md) - Enable extra development-only checks to surface common bugs early.
- [Suspense](Suspense.md) - Display a fallback UI until the children of a boundary finish loading.
- [Activity](Activity.md) - Hide and restore the UI and internal state of children, with pre-rendering and selective hydration.
- [ViewTransition](ViewTransition.md) - Animate trees on enter, exit, update, and shared-element changes using the View Transition API.
