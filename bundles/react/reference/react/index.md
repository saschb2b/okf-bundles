# react (core API)

The `react` package: the Hooks, components, and APIs you import from `react`. Grouped as on react.dev.

- [Hooks](hooks/index.md) - The 18 built-in Hooks (`useState`, `useEffect`, `useContext`, `useReducer`, `useMemo`, `useTransition`, and more).
- [Components](components/index.md) - Built-in components: `<Fragment>`, `<Suspense>`, `<StrictMode>`, `<Profiler>`, `<Activity>`, `<ViewTransition>`.
- [APIs](apis/index.md) - Top-level APIs: `createContext`, `memo`, `lazy`, `use`, `startTransition`, `cache`, `act`, and the taint APIs.
- [Legacy APIs](legacy/index.md) - Deprecated or rarely needed APIs kept for compatibility: `Component`, `createElement`, `forwardRef`, `cloneElement`, `Children`, and more.
