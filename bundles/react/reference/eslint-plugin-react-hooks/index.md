# eslint-plugin-react-hooks

The official ESLint plugin that enforces the [Rules of React](/reference/rules/index.md) and surfaces patterns incompatible with React Compiler's memoization. It bundles a set of lint rules covering Hook placement, render purity, immutability, refs, state updates, and compiler configuration.

- [lints](lints/index.md) - The individual lint rules, one concept per rule.
