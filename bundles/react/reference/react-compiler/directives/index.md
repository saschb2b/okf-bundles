# React Compiler Directives

Source-level directives that opt individual functions into or out of React Compiler optimization. For an overview, see [the React Compiler guide](/react-compiler/react-compiler.md).

- [use memo](use-memo.md) - Marks a function for React Compiler optimization.
- [use no memo](use-no-memo.md) - Prevents a function from being optimized by React Compiler.
